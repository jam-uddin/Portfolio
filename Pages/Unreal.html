Zombie Desolation

Project Overview

"Zombie Desolation" is a survival horror game developed using Unreal Engine. The game challenges players to survive against relentless waves of zombies, utilizing a dynamic point system and various power-ups.

Key Contributions

  • Gameplay Mechanics: Designed and implemented core gameplay mechanics, including a point system that allows players to purchase weapons, upgrades, and essential survival items.
  • AI Systems: Developed multiple zombie AI types with distinct behaviors. For instance, some zombies directly attack the player, while others patrol the environment or steal power-ups.
  • Optimization: Optimized game performance to handle large waves of enemies efficiently, ensuring smooth gameplay even during intense action sequences.
  • Environmental Design: Created immersive environments with realistic lighting and atmospheric effects, enhancing the overall player experience.

Castaway

Project Overview

"Castaway" is a survival adventure game developed using Unreal Engine 5. Players find themselves stranded on a deserted island and must manage resources, fend off wild animals, and find ways to signal for rescue.

Key Contributions

  • Resource Management: Implemented systems for resource collection, crafting, and building, allowing players to create tools, weapons, and shelters essential for survival.
  • Dynamic Environment: Utilized Unreal Engine 5’s capabilities to create a dynamic and visually stunning island environment, complete with day-night cycles and weather changes.
  • AI Systems: Developed AI for wild animals that interact with the player and the environment, adding layers of challenge and realism.
  • Immersive Experience: Leveraged advanced features of Unreal Engine 5 to deliver high-quality graphics and realistic physics, enhancing the immersion and realism of the game world.
